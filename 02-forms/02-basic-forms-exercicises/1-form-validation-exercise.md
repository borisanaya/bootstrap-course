# Formulario con Validación en Bootstrap 5

Este ejercicio práctico muestra cómo crear un **formulario responsive** con **validación nativa de HTML5** y **feedback visual** usando clases de Bootstrap.

---

## Objetivo

Crear un formulario con:
- Validación nativa (`required`, `minlength`, `type="email"`)
- Feedback visual (`is-valid`, `is-invalid`, `invalid-feedback`)
- Diseño responsive con el sistema **Grid** de Bootstrap
- Integración mínima de JavaScript para activar la validación al enviar

---

## Campos del formulario

- **Nombre** (obligatorio, mínimo 3 caracteres)
- **Email** (obligatorio, formato válido)
- **Contraseña** (obligatorio, mínimo 6 caracteres)
- **Checkbox** para aceptar términos (obligatorio)
- **Botón de envío** con validación

---

## Requisitos

- Bootstrap 5.0.2 (CDN)
- Navegador moderno
- Editor de código (VS Code recomendado)
