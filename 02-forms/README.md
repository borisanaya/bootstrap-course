# Descripción del Proyecto: Formularios con Bootstrap

Este repositorio contiene ejemplos y ejercicios prácticos sobre la creación y validación de formularios web utilizando HTML y el framework Bootstrap. A continuación se describe el contenido de cada carpeta principal del proyecto:

---

## 01-basic-forms

Contiene ejemplos básicos de formularios en HTML, mostrando distintas estructuras y elementos de formulario. Es ideal para quienes están comenzando a aprender sobre formularios web.

- **index-01.html** a **index-04.html**: Diferentes variantes de formularios básicos.
- **README.md**: Explicación de los ejemplos incluidos.

---

## 02-basic-forms-exercicises

Incluye ejercicios y soluciones sobre validación de formularios, tanto en HTML puro como con validación en tiempo real.

- **1-form-validation-exercise.md**: Enunciado de un ejercicio de validación de formularios.
- **1-form-validation-solution.md**: Solución propuesta para el ejercicio anterior.
- **1-form-validation.html**: Implementación práctica del ejercicio.
- **2-form-validation-real-time-exercise.md**: Ejercicio enfocado en la validación en tiempo real de los campos del formulario.

---

## 03-sign-in

Ejemplo completo de un formulario de inicio de sesión (sign in) diseñado con Bootstrap 5. Incluye personalización visual y estructura profesional.

- **index.html**: Formulario de inicio de sesión utilizando componentes y utilidades de Bootstrap.
- **signin.css**: Estilos personalizados para complementar Bootstrap.
- **README.md**: Explicación detallada del diseño y la implementación con Bootstrap.
