## Estructura del Repositorio

### 01-starter-template

Plantilla inicial de Bootstrap 5 lista para usar. Contiene un archivo HTML básico con:
- Bootstrap CSS y JS configurados desde CDN
- Estructura HTML5 mínima
- Meta tags necesarios para responsive design

Es el punto de partida ideal para comenzar cualquier proyecto con Bootstrap.

### 02-breakpoint-container-grid

Carpeta de teoría y ejemplos que cubre los conceptos fundamentales de Bootstrap:

- **Containers**: Diferentes tipos de contenedores (`.container`, `.container-fluid`, etc.)
- **Breakpoints**: Puntos de quiebre responsive (sm, md, lg, xl, xxl)
- **Grid System**: Sistema de cuadrícula de 12 columnas

Incluye:
- Guías explicativas en Markdown (README.md, index-01.md, index-02.md)
- Múltiples ejemplos prácticos en HTML
- Demostraciones visuales de cada concepto

### 02-breakpoint-container-grid-exercises

Carpeta de ejercicios prácticos con dos desafíos:

**Ejercicio 1**: Layout responsive básico
- Práctica con headers, sidebars, grids de colores
- Trabajo con columnas responsive
- Control de visibilidad según dispositivo

**Ejercicio 2**: Layout avanzado
- Uso de offsets y order
- Clases de display condicionales
- Reordenamiento de elementos según tamaño de pantalla

Cada ejercicio incluye:
- `readme.md`: Especificaciones del ejercicio
- `start.html`: Archivo inicial para comenzar
- `solution.html`: Solución completa

## Cómo usar este repositorio

1. Comienza revisando la plantilla en `01-starter-template`
2. Estudia los conceptos y ejemplos en `02-breakpoint-container-grid`
3. Practica con los ejercicios en `02-breakpoint-container-grid-exercises`

## Requisitos

- Navegador web moderno
- Editor de código (VS Code recomendado)
- Conexión a internet (para cargar Bootstrap desde CDN)
